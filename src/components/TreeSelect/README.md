# 树下拉框

基于 antd v4 的树下拉框组件 TreeSelect，在多选模式下，新增虚拟选项【全部】

## 功能迭代

### 2020/08/08

1. 多选模式下，新增【全部】选项
2. 允许传入【全部】选项自定义值
3. 允许自定义显示/隐藏【全部】选项
4. 允许保留勾选【全部】选项前的选择值
5. 推荐使用 treeData 传入数据，而不是以 TreeNode 的形式进行渲染


开发中的交互疑问：
1. 当已经勾选了几条数据后，再勾选【全部】然后取消勾选【全部】，此时输入框显示为空数据，还是之前的那几条数据
2. 【全部】是否是固定在最上方，不跟随数据进行滚动
3. 【全部】选项是否存在鼠标滑过的样式，鼠标可点击的范围是多大
